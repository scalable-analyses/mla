Submission Guidelines
---------------------

- Your source code and the sources of your project report must be hosted in a Git repository:
    - Public, or
    - Private (ensure the repository is accessible to us).
- Use Moodle for submissions.
- Every team member must submit! No submission means 0 points, no exceptions!
- Upload a single .tar.xz archive to Moodle which contains:
    - `git_link.txt`: Contains a link to your Git repository.
    - `git_tag.txt`: Contains the Git tag of the submission.
    - Current project report as:
        - `report_link.txt`: Contains a link to GitHub Pages, or
        - `report/` directory: Contains the current project report as HTML.
    - `contribution.txt`: Briefly describes the contributions of every team member.
